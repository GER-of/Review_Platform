# Сайт отзывов о курсах

Платформа для сбора и просмотра отзывов об образовательных курсах.

## Функционал

### Страницы:
1. **Главная страница** (`/`) - список всех курсов с названием, платформой и средней оценкой
2. **Страница курса** (`/course/<id>/`) - детальная информация о курсе, описание и все отзывы
3. **Рекомендуемые курсы** (`/recommended/`) - курсы с оценкой выше 4.5

## Установка и запуск

### 1. Установите зависимости:
```bash
pip install -r requirements.txt
```

### 2. Создайте миграции базы данных:
```bash
python manage.py makemigrations
python manage.py migrate
```

### 3. Создайте суперпользователя для админ-панели:
```bash
python manage.py createsuperuser
```

### 4. Запустите сервер:
```bash
python manage.py runserver
```

### 5. Откройте в браузере:
- Главная страница: http://127.0.0.1:8000/
- Админ-панель: http://127.0.0.1:8000/admin/

## Структура базы данных

См. файл `ER_DIAGRAM.md` для подробной ER-диаграммы.

### Модели:

**UserProfile (Профиль пользователя)**
- user - связь с пользователем (OneToOne)
- bio - описание профиля (до 500 символов)
- avatar - аватар пользователя (изображение)
- created_at - дата создания

**Platform (Платформа)**
- name - название платформы (Coursera, Udemy и т.д.)
- description - описание платформы
- website - сайт платформы
- created_at - дата создания

**Course (Курс)**
- name - название курса
- platform - связь с платформой (ForeignKey)
- description - описание курса
- course_url - ссылка на курс
- created_by - создатель курса (ForeignKey на User)
- is_approved - статус одобрения (Boolean)
- created_at - дата создания

**Review (Отзыв)**
- course - связь с курсом (ForeignKey)
- user - связь с пользователем (ForeignKey)
- author_name - имя автора
- rating - оценка (1-5)
- text - текст отзыва
- created_at - дата создания

## Добавление данных

### Для администраторов:
1. Перейдите на http://127.0.0.1:8000/admin/
2. Войдите с учетными данными суперпользователя
3. Добавьте платформы (Coursera, Udemy, Stepik и т.д.)
4. Модерируйте курсы, предложенные пользователями

### Для пользователей:
1. Зарегистрируйтесь на сайте
2. Нажмите "➕ ДОБАВИТЬ КУРС" в навигации
3. Заполните форму с информацией о курсе
4. Дождитесь одобрения администратором
5. Оставляйте отзывы на одобренные курсы

## Особенности

- **Регистрация и авторизация** - система пользователей с профилями
- **Кастомизация профиля** - загрузка аватара и добавление описания
- **Профиль пользователя** - личная страница с историей всех отзывов
- **Аватары в отзывах** - отображение аватара пользователя рядом с отзывом
- **Добавление курсов пользователями** - зарегистрированные пользователи могут предлагать курсы
- **Модерация курсов** - курсы требуют одобрения администратора перед публикацией
- **Фильтрация по платформам** - на главной странице можно фильтровать курсы по платформам
- **Поиск курсов** - живой поиск по названию курса
- **Ссылки на курсы** - прямые ссылки на страницы курсов на платформах
- **Рекомендации** - автоматическая выборка курсов с высокими оценками
- **Форма отзывов** - авторизованные пользователи могут оставлять отзывы
- **Валидация** - проверка корректности введенных данных
- **Уведомления** - сообщения об успешных действиях
- **Загрузка изображений** - поддержка аватаров пользователей
